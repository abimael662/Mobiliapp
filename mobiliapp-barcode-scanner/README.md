# mobiliapp-barcode-scanner

Custom native barcode scanner plugin with ML Kit integration for Mobiliapp

## Install

To use npm

```bash
npm install mobiliapp-barcode-scanner
````

To use yarn

```bash
yarn add mobiliapp-barcode-scanner
```

Sync native files

```bash
npx cap sync
```
